README FILE FOR HOTFIXES

Deutsch: Siehe weiter unten

English:
NOTE: be careful with replacing files if you have made custom changes!
Use a diff program!

INSTALL: To apply the hotfix(es) contained in this archive, copy the
whole content of this archive into your Contrexx installation. Then
point your browser to your website and add /hotfix to it (example:
http://yoursite.com/hotfix). As you see, the hotfix system will show
you which hotfixes were applied. After this, the hotfix installation is
complete and you may delete the hotfix folder and the README.txt file.

Deutsch:
ACHTUNG: Seien Sie vorsichtig beim ersetzen der Dateien, falls Sie eigene
Anpassungen im Code gemacht haben! Benutzen Sie ein Diff-Programm!

INSTALLATION: Um den/die Hotfix(es) in diesem Archiv zu installieren,
bitte den gesamten Inhalt dieses Archives in das Contrexx-Verzeichnis
kopieren. Danach gehen Sie mit Ihrem Browser auf die URL ihrer
Contrexx-Webseite und fügen hinten /hotfix hinzu (Beispiel:
http://ihreseite.com/hotfix). Wie Sie sehen, zeigt ihnen das System,
welche Hotfixes eingespielt wurden. Danach ist Ihr System aktualisiert,
Sie können den Hotfix-Ordner und die README.txt-Datei in Ihrem
Contrexx-System wieder löschen.
