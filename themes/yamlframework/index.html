Modernes Weblayout verständlich erklärt

"Yet Another Multicolumn Layout" (kurz YAML) ist ein (X)HTML/CSS Framework zur Erstellung moderner und flexibler Layouts auf Grundlage von float-Umgebungen. Dabei stehen ein möglichst hohes Maß an Flexibilität für den Webdesigner und Zugänglichkeit für die Nutzer im Vordergrund. Innerhalb eines Tutorials wird die Funktionsweise der einzelnen Bausteine erläutert.

YAML Version 2.5.2

Die aktuelle Version liefert die vollständige Unterstützung des Internet Explorer 7.0, der seit dem 18. Oktober verfügbar ist. Das seit der Version 2.5 implementierte IE-Clearing musste gegenüber dem IE7 RC1 nochmals leicht überarbeitet werden und arbeitet jetzt fehlerfrei mit der finalen Version des neuen Microsoft Browsers zusammen. Das YAML-Framework unterstützt damit eine umfassende Unterstützung moderner Webbrowser auf verschiedensten Plattformen.

Weitere Korrekturen betreffen Kompatibilitätsverbesserungen im Internet Explorer bei den Listennavigationen sowie Anpassungen der Grundbausteine. Eine vollständige Übersicht der Änderungen der Version 2.5.2 findet sich wie immer im Changelog.

Die aktuelle Fassung des YAML-Frameworks bildet mit diesem Stand auch die Basis für das im Dezember erscheinende Buch "CSS Layouts" beim Galileo Verlag.
