Neue Eigenschaften/Verbesserungen:
----------------------------------

Sicherheitslücken
XSS- und SQL Injection Bugs behoben

Statistik
Referers werden nun mittels JavaScript herausgelesen. Dies optimiert die Statistik bei gecachten Seiten.
Die Statistik-Auswertung wurde in der Genauigkeit ( Zwei Stellen nach dem Komma ) verbessert.

Suche
Wörter mit Umlauten werden nun auch durchsucht.
Suche nach Wortfragmenten erweitert.
Die Gallery wird nun auch durchsucht.

Web Installer
Aus Sicherheitsgründen wird keine "version.xml" Datei mehr erstellt.
Der FTP-Pfad wird nun auch richtig erkannt, wenn der FTP-Server automatisch in ein Verzeichnis wechselt, nachdem man sich eingelogt hat
Die Funktion zur Überprüfung auf eine neuere Version wurde verbessert.

Kontaktformulare
Attachment Dateitypen werden nun überprüft und die Email Benachrichtigungen übersichtlicher mit Tabulatoren dargestellt.
Neu kann mittels hidden Input Tag (Subject), der Email Titel selbst gewählt werden.

Wysiwyg-editor
Version 2.0 des FCKeditors eingebunden und damit viele Fehler verbessert und einige neue nützliche Eigenschaften eingebaut.


Media manager Backend
Die einzelnen Archive (Archive1-3, Content, Themes) sind jetzt in einzelne Bereiche unterteilt. -> Neuer Navigationspunkt "Bilderverwaltung".
Der Mediapfad wird nun auch beim Ausschneiden oder Einfügen von Dateien angezeigt.

Media Manager Frontend
Die einzelnen Archive sind nun mittels eines index.php Scriptes durch das unerlaubte Ansehen der Ordnerstruktur bei falsch konfigurierten Hostings besser geschützt.

Navigation/Menüs
Inaktive Seiten werden auch bei Abruf nicht mehr in der Frontend Navigation dargestellt.
Die ganze Menüstruktur eines beliebigen Frontend Menüs (Dynamisches) kann nun komplet in der navbar.html resp. subnavbar.html Datei im Designmanager definiert werden. Somit kann ab sofort jede mögliche Navigationsart umgesetzt werden.

News
Das Datum kann nun editiert werden. Die News werden neu nach Datum sortiert.

Shop
Bei der Zahlungsart Saferpay kann nun definiert werden welche Kartentypen erlaubt sind.
Der Fehler beim Abspeichern von Änderungen beim Editieren einer Bestellung wurde behoben.

Newsletter
Das Pfad-Offset Problem beim Headlines Generator wurde behoben.
